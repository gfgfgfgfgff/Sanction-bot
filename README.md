# Bot Sanction Discord

Bot de modération avec système de sanctions.

## Commandes
- `/tempmute` - Mute temporaire
- `/unmute` - Retire le mute
- `/to` - Timeout utilisateur
- `/unto` - Retire le timeout
- `/ban` - Bannir utilisateur
- `/unban` - Débannir utilisateur
- `/warn` - Avertir utilisateur
- `/delsanction` - Voir/Supprimer sanctions

## Déploiement
1. Cloner le repo
2. Configurer les variables d'environnement
3. Déployer sur Railway-bot
